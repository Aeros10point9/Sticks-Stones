# This is the repository for Sticks & Stones Files

Please read the following for downloads and instructions on how to get the zip files setup to access the server.
**You need to contact the server owner for the final step!**

## Downloading the Zip

To access the Sticks & Stones server please go and download the zip file **sticks&stones**. This is the zip file to upload to curseforge for setting up a local profile. For guides on how to accomplish this review guides via "how to use zip file to create a profile in curseforge". If you really hit a roadblock or have issues reach out to the server owner.

## Opening the Profile

Once the Zip file has been opened via curseforge and profile created open the profile and check if it loads. If it doesn't please reattempt the Zip file download, if it still has issues please reach out to server owner.

## Accessing the Server

Contact server owner for final step. They will give you a key.

## Updates

This server will go through periodic patches and updates. **Please be aware that if you have server connectivity issues review if you need to recommit to a new Zip download and follow the server access steps again.**

## Feedback
**This is a work in progress, and if you find a bug, issue, or something that needs re-balancing please reach out to the server owner, thank you!**
