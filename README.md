# This is the repository for Sticks & Stones Files

Please read the following for downloads and instructions on how to get the zip files setup to access the server.
**You need to contact the server owner for the final step!**

## Downloading the Zip & .jar file

To access the Sticks & Stones server please go and download the zip file **sticks&stones** and the **tacz-neoforge-1.21.1-1.1.8-hotfix-r3.jar** file. The zip file is the one to upload to curseforge for setting up a local profile. For guides on how to accomplish this review guides via "how to use zip file to create a profile in curseforge". If you really hit a roadblock or have issues reach out to the server owner. As for the jar file, this needs to be added to mods folder in the newly created curseforge profile **PRIOR** to booting the profile for the first time. To do this, nagivate to the profile and open it, right next to the **Play** button should be three dots, click this. There should be an option to **Open Folder**, click this, and within this folder should reside a **mods** folder, this is where you'll place the .jar file.

## Opening the Profile

Once the Zip file has been opened via curseforge and profile created along with placing the .jar file in the mods folder, open the profile and check if it loads. If it doesn't please re-attempt the Zip file download and jar file placement in the mods folder, if it still has issues please reach out to server owner.

## Accessing the Server

Contact server owner for final step. They will give you a key.

## Updates

This server will go through periodic patches and updates. **Please be aware that if you have server connectivity issues review if you need to recommit to a new Zip download and follow the server access steps again.**
